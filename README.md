# Soumi Saha Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript showcasing Soumi Saha’s education, skills, experience, projects, achievements, and contact details.

## Overview

This portfolio site is designed as a modern one-page landing experience with a dark glassmorphism theme, smooth scrolling navigation, and interactive UI elements.

## Features

- Responsive navigation bar with mobile hamburger menu
- Animated typewriter hero text
- Scroll reveal animations
- Active section highlighting in the navbar
- Copy-to-clipboard interaction for contact details
- Downloadable resume link
- Social links to GitHub and LinkedIn

## Files

- `index.html` — main portfolio page structure
- `style.css` — styling and responsive layout
- `script.js` — interactive behavior and page animations
- `profile.jpg` — profile image displayed in hero section
- `Soumi_Saha_Resume.pdf` — downloadable resume file

## Setup

1. Clone or download the repository.
2. Open `index.html` in your browser.

> No build tools or dependencies are required.

## Notes

- Make sure `profile.jpg` and `Soumi_Saha_Resume.pdf` are present in the same folder as `index.html` for the full experience.
- The site uses Google Fonts and Font Awesome from CDNs.
